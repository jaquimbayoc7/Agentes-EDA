# Skill: Generación de reportes

## Descripción
Generación de informes en Markdown y PDF con weasyprint.

## Ubicación
`src/skills/report_builder.py`

## Funciones
- Generación de Markdown con 12 secciones
- Conversión Markdown → PDF con weasyprint
- Generación de decision.json
- Guardado de state_final.json

## Las 12 secciones
§1 Pregunta · §2 Literatura · §3 Hipótesis · §4 Dataset
§5 Preprocesamiento · §6 EDA tabular · §7 EDA temporal
§8 Decisión + H1/H2/H3 · §9 Modelos · §10 Hiperparáms
§11 Advertencias · §12 Próximos pasos + refs
